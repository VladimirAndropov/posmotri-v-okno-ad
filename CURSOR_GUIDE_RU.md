# Руководство по использованию Cursor для фронтенд-разработки

## **1. AI Чат и помощь в коде**

**Задавайте конкретные вопросы:**
- "Добавь адаптивное меню навигации на эту страницу"
- "Создай кнопку переключения темной темы"
- "Исправь CSS для мобильных устройств"
- "Добавь валидацию формы поиска"

**Используйте упоминания @:**
- `@index.html` — ссылайтесь на конкретные файлы
- `@styles/style.css` — работайте с конкретными стилями
- `@scripts/script.js` — изменяйте JavaScript файлы

## **2. Генерация и редактирование кода**

**Для вашего текущего проекта:**

**HTML структура:**
- "Добавь секцию футера с ссылками на соцсети"
- "Создай компонент модального окна"
- "Добавь атрибуты доступности к форме"

**CSS стилизация:**
- "Сделай форму поиска адаптивной для планшетов"
- "Добавь эффекты наведения на карточки видео"
- "Создай анимацию загрузки для прелоадера"

**JavaScript функциональность:**
- "Добавь валидацию формы перед отправкой"
- "Реализуй бесконечную прокрутку для карточек видео"
- "Добавь обработку ошибок для API запросов"

## **3. Лучшие практики в Cursor**

**Навигация по файлам:**
- `Ctrl+P` (или `Cmd+P` на Mac) — быстрый поиск файлов
- `Ctrl+Shift+F` — поиск по всем файлам
- Используйте проводник файлов для просмотра структуры проекта

**Редактирование нескольких файлов:**
- Открывайте несколько файлов во вкладках
- Ссылайтесь на файлы используя `@имя_файла` в чате
- Просите изменить связанные файлы вместе (например, "Обнови и HTML, и CSS для кнопки")

**Проверка кода:**
- "Проверь этот код на лучшие практики"
- "Проверь на проблемы доступности"
- "Оптимизируй этот CSS для производительности"

## **4. Функции специально для фронтенда**

**Живой предпросмотр:**
- Используйте встроенный терминал Cursor для запуска локального сервера
- Или используйте расширение Live Server, если доступно

**Отладка:**
- "Почему мой CSS не применяется?"
- "Отлади эту JavaScript функцию"
- "Найди, где этот стиль переопределяется"

**Рефакторинг:**
- "Рефакторь этот код для использования ES6 модулей"
- "Преобразуй это для использования CSS Grid вместо Flexbox"
- "Вынеси это в переиспользуемый компонент"

   ```
   "Добавь функцию истории поиска, которая сохраняет предыдущие поиски"
   ```

   ```

   ```

3. **Улучшение функциональности:**
   ```
   "Добавь поддержку навигации с клавиатуры для формы"
   ```

4. **Оптимизация:**


## **6. Советы для эффективного использования**

- **Будьте конкретны:** Вместо "сделай лучше" говорите "добавь плавные переходы к состояниям наведения кнопок"
- **Показывайте контекст:** Открывайте релевантные файлы перед задаванием вопросов
- **Итерируйте:** Задавайте уточняющие вопросы для улучшения кода
- **Используйте естественный язык:** Описывайте, что вы хотите, а не как это закодировать

## **7. Типичные задачи фронтенда**

**Макет и дизайн:**
- "Выровняй этот div по центру вертикально и горизонтально"
- "Создай адаптивную сетку"
- "Добавь градиентный фон"

**Интерактивность:**
- "Добавь обработчики кликов на все карточки видео"
- "Реализуй фильтр поиска"
- "Добавь плавную прокрутку"

**Производительность:**
- "Ленивая загрузка изображений в карточках видео"
- "Минифицируй этот CSS"
- "Оптимизируй эту JavaScript функцию"

## **8. Горячие клавиши**

- `Ctrl+K` (или `Cmd+K`) — Быстрое AI редактирование (встроенное)
- `Ctrl+L` — Открыть чат
- `Ctrl+I` — Режим композитора (редактирование нескольких файлов)
- `Tab` — Принять AI предложения
